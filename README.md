# 📱 Job Profile Hub App

A simple Android app that helps you manage and access all your job platform profiles in one place.

---

## 🚀 Features

* 🔗 Add and save profile links from:

  * LinkedIn
  * Naukri
  * Indeed
  * Instahyre
  * Wellfound
* 💾 Persistent storage using SharedPreferences
* 🌐 One-tap open links in browser
* ✏️ Editable fields for updating links anytime
* 🎯 Clean and simple UI

---

## 🛠️ Tech Stack

* Kotlin
* Jetpack Compose
* Android Studio
* SharedPreferences

## ⚙️ How It Works

1. Enter your profile links in the input fields
2. Click **Save** to store them
3. Tap **Open** to launch the link in your browser

---

## 📦 Installation

1. Clone the repository
2. Open in Android Studio
3. Run the app on emulator or device
